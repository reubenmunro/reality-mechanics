# Minimal Ark

The Minimal Ark is the first participant-facing proof nested within the Atlas.

It is not a website, document viewer, graph explorer, AI surface, or media experience. It is a small local environment where a participant can enter one structure-only Atlas term, move to a related term, and see symbolic inhabitants respond.

The structural ratio is:

```text
Ark : Atlas
```

The Atlas is the world and invariant structure. The Ark is the vessel/environment that carries participants through that structure.

## Boundary

The Ark consumes:

- Atlas structure
- Runtime Observatory
- Participant State
- Symbolic Inhabitants

The Ark carries participant-local movement, memory, inhabitants, and symbolic responses.

It does not redefine Atlas structure.

It does not inspect Atlas prose.

It does not bypass runtime layers.

It does not own Atlas ecology. Place-level accumulated traversal, visibility, pressure, and path history belong to Atlas ecology; participant-local experience belongs to the Ark.

```sh
node ark-minimal/server.mjs
```
