# Reference Frame

Reference Frame names the nested placement through which a ratio, relation, scale, or relative read becomes answerable.

A reference frame is not an observer added onto reality. The observer is already nested. A reference frame is the held placement that lets a comparison be checked: from what reference, against what Other, through what boundary, and within what closure scope?

Without a reference frame, ratio can look like a free-floating proportion, scale can look absolute, and relative can be mistaken for arbitrary. With a reference frame, the read becomes testable: change the frame and see what remains invariant, what changes, and what no longer carries.

Reality remains continuous. Different nested reference frames illuminate different faces of the same carrying. Seed may read as latent, held, and shaded from one frame; from another, the same seed may read as carrying, generative, and radiant. The seed has not changed. The frame has.

## Places

Reference Frame places the nested reference through which a ratio, relation, scale, or relative read becomes answerable and one face of carrying becomes readable.

## Holds

Reference Frame is held by [[Ratio]], [[Boundary]], [[Closure Scope]], and [[Readability]]. A comparison must be readable, located, scoped, and followable before its reference frame can be named.

## Pairs

Reference Frame pairs with [[Relative]]. Reference Frame names the placed reference; Relative names the read made answerable through that placement.

## Nests

Reference Frame nests around relative, scale, measure, and reasoning reads as the placement that keeps comparison from pretending to come from nowhere.

## Reads

Reference Frame becomes readable where the reference of a read is explicit enough that change of frame, scale, or scope can be tested rather than smuggled into the conclusion.

It also becomes readable where a changed frame reveals a different face without requiring reality itself to split. The same carrying may be latent from one placement and radiant from another.
