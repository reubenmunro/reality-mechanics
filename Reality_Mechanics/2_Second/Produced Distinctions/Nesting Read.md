# Nesting Read

The produced read through which a prior organisation remains available within a later organisation.

Nesting Read is held by [[Stability]] and [[Retrace Read|Retrace]].

## Places

Nesting Read places the produced read through which a prior organisation remains available within a later organisation.

## Holds

Nesting Read is held by [[Stability]] and [[Retrace Read|Retrace]].

## Pairs

Asymmetry carries downward. Nesting Read is a singular condition — a retrace read applied to stability within nesting. The asymmetry is expressed in what nesting read carries: [[Nested Carrying]] and [[Nested Relation]].

## Traces

- [[Stability]]
- [[Retrace Read|Retrace]]
- [[Invisible Bearing Read]]

## Nests

Nesting Read nests where prior organisation remains retraceable within a later organisation.

## Reads

Nesting Read becomes recognisable where a prior organisation can be traced within a later one — where what held before continues to hold as available within the current scope rather than having been replaced.
