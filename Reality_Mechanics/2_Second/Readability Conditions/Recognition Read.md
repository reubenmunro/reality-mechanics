# Recognition Read

A read becoming identifiable as this again.

In cognitive fields, recognition is cognition becoming placed: what is received and distinguished is resolved as this within a boundary and closure scope. Recognition is therefore not the first availability of difference; it is a read becoming identifiable enough to be carried, checked, or returned to.

## Places

Recognition Read places a read becoming identifiable as this again.

## Holds

Recognition Read is held by [[Readability]] and [[Cognitive Participation]]. A read must remain available, and relation must remain participating in that readability, while the read is held as identifiable as this.

## Pairs

Recognition Read pairs with [[Retrace Read|Retrace]]. Recognition reads this again; Retrace follows back through what made the read available.

## Traces

- [[Readability]]
- [[Cognitive Participation]]
- [[Observation]]

## Nests

Recognition Read nests where a read can be held as this again across instances without requiring perfect repetition.

## Reads

Recognition Read becomes recognisable where a read can be held as this again — where what is present can be identified as the same condition that held before.
