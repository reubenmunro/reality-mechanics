# Second Order

Second Order names the register where [[Carry]] is enacted as [[Carrying]] and connection enters coupling, presence, bearing relation, and participation. First Order makes connection directionally available; Second Order is the active forward movement through it.

In the tree read, Second Order is the trunk. It is the stable carrying body through which root conditions become able to support branching participation.

## Places

Second Order places trunk stability: carrying, coupling, closure, presence, readability, and participation holding enough for later branches to grow without replacing the root structure.

## Holds

Second Order is held by [[First Order]]. First-order connection and carry must be available before carrying can enter participation.

## Pairs

Asymmetry carries downward. Second Order is a structural level marker. The asymmetry it names is fully expressed by the level it carries into: [[Third Order]].

## Reads

Second Order becomes recognisable where connection is actively carrying through coupling, presence, bearing relation, closure, readability, and participation.

It is read as trunk rather than branch. When a later field or domain drifts, the branch can often be pruned or regrown while the second-order carrying structure remains stable.
