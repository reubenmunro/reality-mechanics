# Root System

Root System names the protected source condition from which the Reality Mechanics system is organised.

It protects source structure without becoming produced content.

This note defines the protected condition that keeps the system anchored to its dependency path.

The root system is not the primitive itself. It protects the primitive and the order through which the system remains readable.

## Places

Root System places the protected source condition from which the Reality Mechanics system is organised.

## Holds

Root System is held by [[Relation]], [[Root Order]], and [[Invariant]].

## Pairs

Root System is vertical. It names the protected source condition from which the Reality Mechanics system is organised, and carries this forward into [[Root Resilience]]. Root System holds [[Relation]], [[Root Order]], and [[Invariant]] — it is the foundational source condition, not a term that sits beside a lateral co-present sibling. Root Resilience holds Root System and is therefore downstream, not co-present.
