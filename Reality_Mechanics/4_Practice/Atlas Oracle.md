# Atlas Oracle

Atlas Oracle names the bounded instrument by which a participant brings pressure to the [[Atlas]], receives an Atlas-grounded answer, inspects the trace, and can challenge what does or does not hold.

It is not a claim that AI knows the truth. It is a way for AI to answer through the Atlas while leaving enough trace for the answer to be accepted, rejected, corrected, or tended.

The simple human surface is:

```text
Ask the Atlas.
Receive an answer.
Inspect the trace.
If it does not hold, trace why.
```

The answer is provisional. The trace is inspectable. Disagreement is participation.

The Oracle is available only while its [[Oracle Carrying Conditions]] hold. If those conditions fail, the Oracle pauses rather than drifting into unbounded AI use.

## Places

Atlas Oracle places a bounded interface where pressure can enter as a question, confusion, challenge, absence, or need for orientation, and leave as an answer that remains answerable to its Atlas trace.

## Holds

Atlas Oracle is held by [[Atlas]], [[Reasoning]], [[AI Participation]], [[Ark Run]], [[Check]], [[Generate and Regenerate]], [[Carried Condition]], and [[Oracle Carrying Conditions]].

The Atlas provides the dependency-ordered surface. Reasoning keeps the answer answerable as traceable support. AI Participation constrains the artificial participant. Ark Run carries pressure through trace, bound, determination, and next posture. Check lets the answer be challenged. Generate and Regenerate tests whether the trace can be continued. Carried Condition holds the return point where load exceeds solitary carrying and meaning can wait. Oracle Carrying Conditions keep governance inside the operation.

## Pairs

Atlas Oracle pairs with open-ended chat.

Open-ended chat may continue fluently without preserving the carrying path. Atlas Oracle carries one pressure through a bounded answer and exposes enough trace for the participant to test whether it holds.

## Nests

Atlas Oracle nests inside [[Atlas Practice]] as a human-facing instrument and AI-facing bounded operation.

For a human participant, it is a doorway: bring a pressure, receive an answer, inspect the trace, and challenge the movement.

For an AI participant, it is an operation:

```text
input pressure
-> carrying conditions check
-> MCP entry and traversal
-> Atlas-grounded answer
-> trace path
-> bound and uncertainty
-> next recarry
-> review event
```

## Reads

Atlas Oracle becomes recognisable where the value is not that AI is always right, but that its move can be traced.

If the answer holds, the participant can continue from it.

If the answer does not hold, the challenge becomes tending:

```text
wrong handle
missing trace
too much interpretation
bound crossed
answer did not carry
```

The review is not only feedback on AI quality. It is a trace event that can show where the Atlas needs tending, where a route is missing, where a handle is overloaded, or where the answer tried to carry more than the trace could hold.

Atlas Oracle stores trace events by default, not private stories:

```text
pressure category
selected handles
trace path
answer bound
review action
raw personal text stored: false
```

The Oracle does not answer merely because a question arrived. It answers only where the carrying conditions hold:

```text
bounded input
bounded answer
MCP trace
structured output
review path
cost cap
rate limit
storage bound
safety route
pause available
```

The Oracle may be embedded in other places only where the Atlas trace remains attached:

```text
source visible
trace visible
review visible
bound visible
no diagnosis claim
no final authority claim
raw storage off by default
```

An embed is not a separate Oracle. It is the same Oracle carried in another place.

Where koha is offered, it should come after value has been carried:

```text
If this carried value for you, koha helps carry it forward.
```

Koha is not a paywall. It is a return path for shared carrying.

Where load is too concentrated for interpretation, Atlas Oracle should return through [[Carried Condition]] before meaning:

```text
You are carried.
Meaning can wait.
```
