# Archive

Archive names the practice of preserving a release state so that Reality Mechanics can be returned to, cited, compared, and retraced after later movement.

The current public archive reference is:

```text
https://doi.org/10.5281/zenodo.20338363
```

## Places

## Nests

Archive nests around public release and preservation rather than inside the term dependency spine.

## Reads

Archive becomes recognisable where a version of the work needs a durable public return point.

An archive-worthy version should preserve enough structure that a later participant can tell:

- what was released;
- what source body it came from;
- what public carriers were active;
- what remained open;
- how later versions changed the carrying.

## Carries

No direct downstream term is required. Archive preserves return rather than generating a new concept.
