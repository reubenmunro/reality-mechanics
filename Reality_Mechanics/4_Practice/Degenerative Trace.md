# Degenerative Trace

Degenerative Trace names the trace of where carrying continued while contact with its generating order was lost.

It is not merely a missing trace. It is the readable path of trace-loss inside continuing carrying.

The core read is:

```text
Degenerative Trace locates where carrying continued after losing contact with what generated it.
```

## Places

Degenerative Trace places the trace of where carrying continued while contact with its generating order was lost.

## Holds

Degenerative Trace is held by [[Trace]], [[Degenerate]], [[Carrying]], [[Source Drift]], and [[Regenerate]].

Carrying must continue. Degenerate must make generative trace loss readable. Source Drift names the pattern where the carrier is treated as source. Regenerate keeps the diagnostic oriented toward return rather than accusation.

## Pairs

Degenerative Trace pairs with [[Generative Trace]]. Generative Trace preserves the generating order strongly enough to seed continuation; Degenerative Trace follows where carrying continued while that generating order was lost.

## Nests

Degenerative Trace nests inside [[Atlas Practice]] as the diagnostic trace used when carrying still appears to continue but its source, dependency path, or generative order has become unrecoverable.

It does not replace [[Degeneration Read]]. Degeneration Read locates the broader path from healthy carrying through strain, degeneration, and repair. Degenerative Trace names the practice trace that follows where the generating order was lost.

## Reads

Degenerative Trace becomes recognisable where a participant can ask:

```text
What kept carrying?
What source or dependency became unrecoverable?
Where did the generated form begin behaving as generator?
What pressure, speed, overload, simplification, extraction, or drift hid the loss?
What trace remains available for regeneration?
```

The point is not to prove that a carrying is bad. The point is to locate where continuation stopped being answerable to what generated it, so regeneration can return through a real path rather than inventing a replacement.
