# Thought

Thought names carried read inside cognition: readable relation modelled, named, interpreted, projected, questioned, or rehearsed.

A thought is a carried read or model. It is not automatically true, complete, or compatible.

Thought does not create order by itself. It becomes readable where relation has enough [[Clearance]] and [[Readability]] for a pattern to be taken up into cognitive form.

## Places

Thought places the condition through which readability is modelled or named — the carrying of a recognised pattern into a form that can guide attention or action.

## Holds

Thought is held by [[Cognition]], [[Read]], [[Readability]], [[Recognition Read|Recognition]], [[Pattern]], and [[Medium]]. Readability must be present and pattern must be recognisable before the modelling or naming of that pattern can be carried as thought.

## Pairs

Thought pairs with [[Memory]]. Thought names the current cognitive carry; Memory names thought that recurs and remains retraceable.

## Nests

Thought nests within the Cognition Field as the condition through which readability is carried into nameable form — the structural ground for memory, self, and cognitive carrying reads.

## Reads

Thought becomes readable where relation is taken up into a cognitive model that can be held, revised, spoken, checked, or released.

This means thought is answerable to what made it readable. A thought may model order, but it does not become grounded until it can return to the relation, body, field, or source that carries it.

The dependency movement is:

First: [[Availability]] and [[Boundary]] make something distinguishable enough to be taken up.

Second: [[Read]], [[Readability]], [[Recognition Read|Recognition]], [[Pattern]], and [[Medium]] make modelling, naming, interpretation, and projection evaluable.

Third: [[Thought]] places cognitive modelling inside [[Cognition Field]] with bridge pressure from language, expression, and relation.

Higher-Order Edge:

Thought remains third-order where it names a cognitive model, interpretation, projection, question, or rehearsal.

It crosses into higher order where [[Consciousness]] makes thought readable as thought and [[Self-Modelling]] or [[Self-Regulation]] changes how that thought is carried.

Movement:

```text
readability becomes available
-> cognition models or names it
-> thought can guide attention or action
-> truth and compatibility still need checking
```
