# Cognitive Reads

Cognitive reads name branch-level reads concerned with pre-explicit availability, memory, retained readability, operation on readable order, discernment, wisdom, and recursive self-readability.

## Places

Cognitive Reads places the set of path reads through which thought and memory become evaluable at a closure scope — the branch of reads that apply cognitive structural conditions to specific passages.

## Holds

Cognitive Reads is held by [[Cognition]], [[Read]], [[Readability]], and [[Closure Scope]]. Cognition must hold and readability must be present before cognitive path reads can be named.

## Pairs

Cognitive Reads pairs with [[Cognitive Carrying]]: cognitive reads names the set of path reads through which thought and memory become evaluable at a closure scope; cognitive carrying names the condition through which a cognitive state is continued through participation. Each requires the other — reads presuppose something being carried; carrying presupposes reads through which what is carried can be evaluated.

## Reads

Cognitive Reads becomes readable where a condition becomes available to cognition as something that can be evaluated, retained, discerned, remembered, or re-entered within a closure scope.

The dependency movement is:

First: [[Availability]] makes a distinguishable condition present enough to participate.

Second: [[Readability]] and [[Read]] make that condition evaluable at [[Closure Scope]].

Third: [[Cognitive Reads]] names cognition-specific use of those produced reads.
