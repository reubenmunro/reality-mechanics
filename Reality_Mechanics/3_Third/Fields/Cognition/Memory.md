# Memory

The condition through which a carried read becomes returnable — a read that has been borne through thought and remains available to adjust future carrying.

Memory is held by [[Cognition]], [[Thought]], [[Recurrence]], [[Carrying]], and [[Retrace Read|Retrace]]. Thought must carry a read and recurrence must make it returnable before the carried read can be named as memory.

Storage can carry memory, but storage alone may preserve information without preserving the bearing path that made the read matter. Memory becomes structural where a carried read can be re-entered without losing enough of its dependency path to keep participation answerable.

## Reads

Memory becomes recognisable where a carried read remains available for return, comparison, correction, or future participation without becoming detached storage.

Carrying Modes:

Memory has the same structural condition in humans, computers, and the Atlas: a returnable carried read. What differs is the mode of carrying.

Organic memory is returnable carried read arising through embodied, cognitive, relational, affective, or recurring participation.

Engineered memory is returnable carried read maintained through designed storage, addressing, files, logs, models, versions, or systems.

Hybrid memory is returnable carried read carried across organic and engineered participation.

These are classifications of carrying mode, not separate substances or separate realities.

The dependency movement is:

First: [[Availability]] and [[Boundary]] make a read distinguishable enough to be held.

Second: [[Carrying]], [[Recurrence]], and [[Readability]] make that read returnable.

Third: [[Memory]] places returnable carried read inside [[Cognition Field]].

Higher-Order Edge:

Memory remains third-order where it names returnable carried read.

It crosses into higher order where [[Consciousness]], [[Recursion]], or [[Higher Learning]] changes how memory is carried, corrected, or used to alter future participation.

Movement:

```text
read becomes carried
-> thought
-> carried read becomes returnable
-> memory
-> returnable read can adjust future carrying
-> learning
```
