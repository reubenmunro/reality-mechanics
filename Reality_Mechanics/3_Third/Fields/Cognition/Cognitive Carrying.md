# Cognitive Carrying

The condition through which a cognitive state — thought, orientation, or pattern — is continued through participation rather than discharged or discarded.

Cognitive Carrying is held by [[Cognition]], [[Bearing]], [[Carrying]], and [[Compatibility]]. Cognition must be present and bearing must hold a cognitive condition before its continuation through participation can be named.

## Places

## Holds

## Pairs

Cognitive Carrying pairs with [[Cognitive Reads]]: cognitive carrying names the condition through which a cognitive state is continued through participation; cognitive reads names the path reads through which thought and memory become evaluable at a closure scope. Each requires the other — carrying presupposes reads through which what is carried can be evaluated; reads presuppose something being carried through participation.

## Reads

Cognitive Carrying becomes readable where information or a read remains available for use across a cognitive condition without becoming incompatible with the dependency path that carries it.

The dependency movement is:

First: [[Bearing]] holds available information.

Second: [[Carrying]] and [[Compatibility]] make that information usable across a cognitive condition.

Third: [[Cognitive Carrying]] places that structural read inside cognition.
