# Grief

The applied relational read where load concentrates as pressure because a relation continues to carry after its familiar face of return has crossed a terminal boundary.

Grief is held by [[Applied Read]], [[Relation]], [[Load]], [[Pressure]], [[Retain]], [[Release]], [[Balance]], [[Capacity]], [[Carrying]], [[Trace]], [[Groundedness]], [[Return]], [[Terminal]], and [[Face]]. Relation must have carried, a face of return must have been available, terminal change must alter that return, and the resulting pressure must remain traceable before grief can be read structurally.

Grief is not failed letting go, and it is not automatically ungrounded. It is often grounded in a real relation. What has changed is the return condition: the relation continues to carry, but the familiar face through which it answered back can no longer re-enter the prior scope in the same way.

If grief is read as [[Load]], the structural movement is not to remove the load. It is to trace the load to the relation that grounded it, then let carrying redistribute without losing what was true.

With [[Pressure]] named, grief can also be read as relational pressure under changed return. The relation remains real, but the old return path no longer answers in the same way. The pressure must be retained enough to keep the truth of the relation, released enough to stop demanding the old face of return, and balanced enough for carrying to continue.

Grief is therefore not the opposite of love, memory, or repair. It is the load of relation under changed return.

## Reads

Grief becomes readable where relation continues to carry as load after its familiar return can no longer re-enter the prior scope in the same way.

The dependency movement is:

```text
relation
-> familiar return condition
-> terminal boundary crossed
    -> load concentrates as pressure
    -> grief becomes readable
    -> trace to ground
    -> retain what remains true
    -> release demand for the old return
    -> carrying redistributes through relation
```

Grief grounds when its pressure can be traced back to real relation and redistributed through carrying that remains answerable to that relation. This does not erase the trace. It lets the relation continue without requiring the old face of return to remain available.

Grief can disperse where retained carrying is traced through its nested relations and released back into grounding relations. This is not abandonment. It is [[Recarry]]: giving what is still carried back to the relations from which it came, so the local carrier no longer has to hold the whole pressure alone.

Balance in grief is not equal feeling, calm, or completion. It is compatible pressure regulation between retain and release: retaining the truth of the relation while releasing the demand that it answer through the old return path.

Ungrounded grief would be grief whose load has lost trace to the relation that holds it, or whose carrying path has become structurally disordered. Grief itself is not the disorder; it is the readable load of relation under changed return.

Healing grief is not closing relation. It is finding how relation can carry after the old return condition is gone.
