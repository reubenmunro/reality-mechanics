# Applied Read

Local use of a path read in a specific case.

Applied Read is held by [[Path Reads]] and [[Closure Scope]].

An applied read tests whether a field or domain translation still holds when it becomes local.

## Places

Applied Read places local use of a path read in a specific case.

It places the check point after a structural term has crossed a [[Translation Boundary]] and become terminal to field or domain language.

## Holds

Applied Read is held by [[Path Reads]] and [[Closure Scope]].

## Pairs

No lateral pair is required at this placement yet; this term currently reads as a branch or terminal read.

## Nests

Applied Read nests where a path read becomes usable in a bounded case without becoming the case itself.

It nests below branch translation: the local case tests whether the terminal domain expression still answers to the structural trace.

## Reads

Applied Read becomes recognisable where a path read can be tested within a specific local condition.

It becomes especially useful where a domain term can be traced back to the Atlas term it translates, then checked against the local case.
