# Enchantment

Altered readability through influence — the condition where influence changes how something is read so that participation is drawn, constrained, heightened, or misread through that altered read.

Enchantment is held by [[Influence]] and [[Readability]]. Influence must be present and readability must be alterable before enchantment can be read as distinct from ordinary influence or ordinary appearance.

## Nests

Enchantment nests within Path Reads as the root condition from which altered-readability reads — Spell, Glamour, Illusion, Binding, Wonder — are derived.

## Reads

Enchantment becomes recognisable where influence changes the readability of a condition so that participation is drawn, constrained, heightened, or misread through that altered read. The path through readability and influence must be entered for enchantment to become readable rather than treated as source power or mere feeling.
