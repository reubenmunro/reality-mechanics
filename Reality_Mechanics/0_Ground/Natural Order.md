# Natural Order

Order already available within the governing arrangement — dependency readability that holds without being imposed by the read that enters it.

Natural Order is held by [[Root Order]].

Natural Order does not require entry by read in order to hold. A read may enter Natural Order, trace it, name it, measure it, act within it, or alter the relation it joins, but the read does not create the order it enters.

## Places

Natural Order places the order already available within the governing arrangement — dependency readability that holds without being imposed by the read that enters it. It names availability rather than imposition.

## Holds

Natural Order is held by [[Root Order]].

## Pairs

Natural Order pairs with [[Dependency Order]]. Natural Order names the order available within the governing arrangement without imposition; Dependency Order names the structural precedence within that same arrangement. Each requires the other: available order is only readable against what must structurally precede it; structural precedence is only nameable against the order it orders.

## Traces

- [[Root Order]]
- [[Ground]]

## Nests

Natural Order nests wherever order holds through available relation rather than continual external correction.

## Reads

Natural Order becomes recognisable where recurring conditions continue to preserve the same dependency structure without requiring continual external correction. It is order holding because it is available, not because it is imposed.

Entering the order as a read is required for the Atlas to trace it, not for the order to exist. Once entered, a read becomes participation and must be checked for whether it remains traceable and compatible with the relation it affects.
