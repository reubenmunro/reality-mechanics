# Read

Recognition of meaningful distinction at the current scope.

Read is held by [[Trace]]. The distinction must remain followable for it to be taken up as meaningful — trace does not finish before read begins; a read may widen or recur without exhausting the trace it draws on.

Read enters order; it does not create order. Once entered, read participates in what it traces: it can clarify, distort, repair, accelerate, name, measure, or alter the relation it has entered. The question is whether the read remains traceable and compatible with what carries it.

## Pairs

Read is vertical. It names recognition of meaningful distinction and carries this forward into [[Readability]], [[Thread]], [[Step]], and [[Retrace Read|Retrace]]. Read holds [[Trace]]; it is downstream of trace in the carrier chain, not co-present with it as a lateral pair.

## Nests

Read nests between trace and readability. It can widen or recur without exhausting the trace.

## Reads

Read becomes recognisable where a traced distinction is taken up as meaningful: the trace is not only followable but significant within the current scope.

Read also becomes recognisable where entry into an already ordered condition changes the relation being read. This makes read a participating condition that must remain answerable to trace.
