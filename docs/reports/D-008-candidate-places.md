## Candidate 1 — Generic primitive *Drift*

### Repeated occurrences

Ungrounded word *drift* in opening prose: **14 terms** (e.g. Clean, Number, Scale, Retrace Practice, Source Drift, Recursion).

Named drift-family terms already exist: **Source Drift**, **Term Drift**, **AI Drift**, **AI Reply Drift**, **Inherited Term Drift**, **Control Drift**, **Harmonic Drift**, **Degeneration Read** (carries source-drift reads).

### Dependency path

Drift reads trace through **Degenerate**, **Higher** path reads, and **practice** drift instruments — not through a missing first-order primitive.

### Retrace path

Any opening that says *drift* without wikilink fails retrace locally — fixed by **Class A refinement** (wikilink or remove), not by inventing a new root term.

### Why existing terms are insufficient

They are **not** insufficient — the Atlas already names multiple drift **places** at appropriate orders. The gap is **prose fidelity**, not ontology.

### Confidence

**High** that no new *Drift* primitive is required.

### Failure mode if omitted

Prose continues to interpret where it should link — Class A backlog item.

**Verdict:** D — language refinement only.
