# D-021.2 — Observatory Landing

**Programme:** Public Structure  
**Type:** Observatory Landing  
**Date:** 2026-07-06 (UTC+12)

---

## Objective

Reorganise the Observatory landing so the first screen orients the visitor before observation begins. The Field canvas and runtime mechanics are unchanged. Atlas and Theory are not duplicated.

---

## Before / after

### Before (live prior to D-021.2)

- Field canvas dominated the viewport immediately.
- Term sheet opened on load with D-020A content for the default focus (e.g. Relation).
- No instrument framing: little sense of *what* the Observatory is before interacting.
- Legacy `#top` header remained in markup (hidden inconsistently).

**Screenshot (before):** Steward capture — `https://realitymechanics.nz/field` prior to deploy; full viewport showing Field + auto-open term panel.

### After (D-021.2)

- Compact landing block above the Field: title, one-line orientation, three actions.
- Neutral Observatory panel before first term selection.
- D-020A compressed term panel unchanged after selection.
- Landing collapses to title-only once observation begins (`body.landing-dismissed`).
- Field canvas begins immediately beneath; no hero, no extra scroll, no marketing copy.

**Screenshot (after):** Steward capture — same URL post-deploy; landing header + neutral panel + Field visible in one viewport.

---

## Landing structure

| Element | Content / behaviour |
|---|---|
| Title | Reality Mechanics Observatory |
| Orientation | Observe how structural relationships become visible through participation. |
| Observe the Field | Dismisses landing copy/actions; keeps neutral panel open |
| Continue where I left off | Shown when `localStorage.observatory.lastFocusId` is valid; calls `enterOperation` |
| Browse the Atlas | Opens canonical GitHub tree (`Reality_Mechanics/`) — no embedded Atlas |

---

## Panel behaviour

| State | Panel |
|---|---|
| No term selected | **Observatory** — *Select a place in the field to observe its structure.* |
| Term selected | D-020A sheet: order, title, place, relations, View Atlas Entry |

Close (×) and re-selecting the focused node return to the neutral panel without hiding the sheet.

---

## Navigation behaviour

- Public nav unchanged from D-021.1 (Observatory · Pulse · Theory · Proof).
- **Observe the Field** — instrument mode; landing actions hidden; neutral panel.
- **Continue where I left off** — restores last observed term via stored focus id.
- **Browse the Atlas** — external link; leaves Observatory.
- Field term selection / enter form / hash deep-link — dismiss landing, open D-020A panel.
- Hash deep-link on load — auto-observes linked term; no hash on first visit until a term is chosen.
