## Steward use

1. Pick one term (or authorised batch) from Tier 1 onward.
2. Run five-pass Structural Reading (`docs/practice/STRUCTURAL_READING.md`).

4. Steward accepts, rejects, or defers before Atlas edit.
5. After Atlas prose commit: D1 sync (`npm --prefix .atlas-publisher run sync:d1 -- --apply`).

Do not promote terms from backlog inference alone.
