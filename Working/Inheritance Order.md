# Inheritance Order (Working)

> Working document. Not canon. This document exists to test whether inheritance order explains existing Atlas structure more faithfully than dependency order alone.

## Distinction

Dependency asks:
- What must already be true?

Inheritance asks:
- Which primitive's mechanic is this family unfolding?

These are not assumed to be identical.

## Test

For each family:

1. Recover the family invariant.
2. Test every member against the generating primitive.
3. If one member fails, determine whether the family, the primitive, or the note has drifted.

No ontology changes should be made until this hypothesis survives repeated structural testing.
